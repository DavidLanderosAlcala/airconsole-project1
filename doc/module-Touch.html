(inner) button(selector, callback)

converts DOM elements into buttons and installs an event listener
Parameters:
Name Type Description
selector string the CSS selector to hook the DOM elements
callback function called for each event ocurred with the buttons

Example
Touch.button(".my-button", function(e) {
    console.log("the button %s has been %s", e.sender.id, e.isPressed ? "pressed" : "released" );
});
